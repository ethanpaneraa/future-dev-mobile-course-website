# Course Assignments

Welcome to the assignments section of our mobile development course! Here you'll find all the weekly assignments that will guide you through building "TaskGram" - a social task management app. Each assignment builds upon the previous one, helping you develop a fully functional mobile application by the end of the course.

## Assignment Schedule

1. [Week 1: Foundation & Environment Setup](/content/assignments/assignment-one)

   - Development environment setup
   - Project initialization
   - Basic navigation implementation
   - Supabase authentication integration

2. Week 2: Database & User Profiles

   - Database schema implementation
   - User profile creation
   - Authentication flows
   - Security implementation

3. Week 3: Task Management & Storage

   - Task creation and management
   - Image upload functionality
   - Camera integration
   - Storage handling

4. Week 4: Real-time Features

   - Real-time data subscriptions
   - Social feed development
   - Friend system implementation
   - Activity monitoring

5. Week 5: Social Interactions

   - Likes and comments system
   - Notification setup
   - Activity feed creation
   - User interaction features

6. Week 6: Polish & Performance

   - UI/UX improvements
   - Performance optimization
   - Error handling implementation
   - Testing integration

7. Week 7: Final Project
   - App completion
   - Deployment preparation
   - Documentation
   - Final presentation

## Submission Guidelines

For each assignment:

1. **Code Submission**

   - Submit via GitHub repository
   - Include clear documentation
   - Follow provided code style guidelines
   - Include setup instructions in README

2. **Demo Requirements**

   - Screen recording of feature implementation
   - Test coverage (when applicable)
   - Performance metrics (when applicable)

3. **Documentation**
   - Implementation details
   - Challenges faced and solutions
   - Future improvements

## Grading Structure

Each assignment will be evaluated based on:

- Functionality (40%)
- Code Quality (25%)
- Documentation (20%)
- Extra Credit Implementation (15%)
